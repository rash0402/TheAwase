# ヘラブナ釣りにおける「アタリ」のメカニズムと分類

## 1. 概要
ヘラブナ釣りにおける「アタリ」は、魚がエサにコンタクトした際にウキに現れる動きの総称です。
主なアタリとして「食い上げ」と「消し込み（ツン）」があり、これらは魚の捕食行動と物理的な力学関係によって発生します。

## 2. アタリの分類とメカニズム

### 2.1 食い上げ (Kuiage / Lift Bite)
ウキが水面から浮き上がってくるアタリ。

*   **現象:** なじみ（エサの重さで沈んだ状態）から、フワッあるいはツツッとウキが上昇する。
*   **物理メカニズム:**
    1.  **テンション抜重:** 魚がエサ（およびハリス・オモリ）を吸い込み、そのまま上方向へ移動する、あるいは単にエサを持ち上げることで、ウキにかかっていた下方向の荷重（エサ＋オモリの重力）が減少・消失する。
    2.  **浮力優勢:** 荷重が減少した結果、ウキの残存浮力が勝り、バランスする位置までウキが上昇する。
*   **発生状況:**
    *   底釣りで頻発する（底にあるエサを拾って上を向くため）。
    *   宙釣りでも、魚がエサを咥えて食い上げる（リフトする）場合に発生。

### 2.2 消し込み (Keshikomi / Plunge Bite)
ウキが水中へ鋭く引き込まれるアタリ。「ツン」「チク」もこの一種だが、特に深く入るものを消し込みと呼ぶことが多い。

*   **現象:** ウキが水中にズバッと強く沈む。
*   **物理メカニズム:**
    1.  **吸引と牽引:** 魚が強い力でエサを吸い込む、または吸い込んだ勢いで反転・潜行することで、道糸に対し強い下向きの引力が働く。
    2.  **慣性力:** 瞬発的な力が加わるため、ウキの浮力と抵抗を打ち破って沈下する。
*   **発生状況:**
    *   魚の活性が高い時（やる気がある時）。
    *   エサをひったくるような強い捕食。

### 2.3 サワリ (Sawari / Pre-bite)
本格的な食いアタリの前兆。

*   **現象:** ウキがフワフワ、ユラユラと不規則に動く。明確な「節」の移動ではない微細な動き。
*   **物理メカニズム:**
    *   魚がエサや糸の近くを遊泳するときに発生する水流（及ぼす水圧）。
    *   魚体が糸やエサに軽く触れる（タッチする）物理的接触。
    *   エサを遠巻きに吸い込もうとする予備動作。

### 2.4 アタリ返し (Atari-gaeshi / Bite on Return)
なじみきったウキが、エサ落ちに伴ってゆっくり戻して（浮き上がって）くる過程で出るアタリ。

*   **現象:** バラケエサが溶けてウキがトップを上げ始めたタイミング（戻し局面）で、鋭い「ツン」や「食い上げ」が入る。
*   **物理メカニズム:**
    *   エサが適度に小さく・軽くなり、吸い込みやすくなった瞬間に魚がコンタクトする。
    *   エサ落ち（抜け）の瞬間の動きに反応して食いつく。

## 3. シミュレーション実装への示唆

### 魚の挙動モデル (Fish AI) への追加要件
現在 (`ATTACK` ステート) の「吸い込み力場」だけでは、食い上げと消し込みの違いを表現しきれない可能性があります。食餌後の**事後動作 (Post-Attack Motion)** を導入すべきです。

1.  **Attack Vectorの多様化:**
    *   吸い込み力の方向ベクトルをランダム、あるいは状況（底釣りか宙釣りか）に応じて変化させる。
    *   **Upward Bias (食い上げ用):** 吸い込みながら上へ移動するベクトル。
    *   **Downward/Lateral Bias (消し込み用):** 吸い込みながら反転・潜行するベクトル。

2.  **Bait Interact Stateの追加:**
    *   「吸い込み中（Holding）」の状態を定義し、その間の魚の移動がダイレクトにハリス・道糸を通じてウキに伝わるようにする。

### 物理モデルへの影響
*   **食い上げ:** エサの質量効果を無効化（魚が支える）するロジックが必要。
    *   魚がエサをHoildingしている間は、エサの重力が道糸に伝わらず、魚の運動（持ち上げ）が支配的になる。
