# 趣味のゲーム開発プロジェクト

これは、Geminiと一緒に作る趣味のゲームプロジェクトです。

## 概要

このプロジェクトの目的は、楽しみながら小さなWebゲームを完成させることです。
ゲームの具体的な内容は、`docs/GDD.md`（ゲームデザインドキュメント）にまとめていきます。

## はじめ方

1.  このリポジトリをクローンします。
2.  `index.html`をブラウザで開くと、現在のゲームの状態を確認できます。

## ディレクトリ構成

-   `assets/`: 画像や音声ファイルなどの素材を置く場所です。
-   `docs/`: ゲームデザインドキュメントなど、設計に関する資料を置く場所です。
-   `src/`: ゲームのメインロジックとなるJavaScriptコードを置く場所です。
-   `index.html`: ゲームを表示するためのメインのHTMLファイルです。
-   `style.css`: ゲーム画面の基本的なスタイルを定義します。
